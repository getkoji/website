# Koji Website

Marketing site for [Koji](https://github.com/getkoji/koji) — the open source document processing platform.

## Stack

- [Astro](https://astro.build) — static site framework
- Vanilla CSS (no framework)
- Deployed to Cloudflare Pages

## Development

```bash
npm install
npm run dev
```

Opens at [http://localhost:4321](http://localhost:4321).

## Build

```bash
npm run build
npm run preview
```

Static output goes to `dist/`.
